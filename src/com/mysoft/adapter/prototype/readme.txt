一。	适配器模式

	1. 概述
		将一个类的接口转换成客户希望的另一个接口，Adapter模式使得原来由于接口不兼容
		不能在一起工作的那些类可以在一起工作
	2. 解决问题
		Adapter模式使得原来由于接口不兼容而不能一起工作的那些类可以一起工作
	3. 模式中的角色
		3.1 目标接口（Target）:客户所期待的接口，可以是具体或者抽象的类，也可以是接口
		3.2 源角色(Adaptee):需要适配的类
		3.3 适配器(Adapter):通过包装一个需要适配的对象，把原接口转换成目标接口
	4. 模式解读
		Java中的适配器模式可以分为：类适配器模式和对象适配器模式
